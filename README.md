# Challenge #1 - Job Finder App UI 🚀

Este proyecto es el primer desafío de desarrollo Android con **Jetpack Compose**. Consiste en la creación de una interfaz de usuario moderna, limpia y navegable basada en un diseño de Figma.

## 📱 Pantallas Implementadas
* **Welcome Screen:** Pantalla de bienvenida con ilustración y acceso principal.
* **Login Screen:** Formulario de ingreso con validación visual de campos.
* **Register Screen:** Registro de nuevos usuarios con confirmación de contraseña.

## 🛠️ Tecnologías y Conceptos Utilizados
* **Kotlin**: Lenguaje principal.
* **Jetpack Compose**: Para la construcción de la UI declarativa.
* **Navigation Compose**: Implementación de `NavHost` para el flujo entre pantallas.
* **Material Design 3**: Uso de componentes, tipografías y colores personalizados.
* **Gestión de Assets**: Importación y optimización de recursos gráficos.

## 🏗️ Arquitectura
El proyecto sigue una estructura limpia, separando las pantallas en archivos independientes para mejorar la mantenibilidad:
* `MainActivity.kt`: Punto de entrada y contenedor principal.
* `AppNavigation.kt`: Enrutador central de la aplicación.
* `LoginScreen.kt` / `RegisterScreen.kt`: Definición de componentes de UI.
